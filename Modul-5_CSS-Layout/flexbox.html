Kesimpulan skenario 1 adalah, dengan menambahkan properti display: flex;, urutan angka yang sebelumnya tersusun ke bawah menjadi tersusun ke samping.

Kesimpulan skenario 2 adalah, ketika kita menambahkan aturan CSS pada elemen anak dari kelas wadah_flex, maka tampilan seperti warna latar belakang, jarak antar elemen (margin), ruang dalam elemen (padding), dan ukuran font akan disesuaikan.

Kesimpulan skenario 3 adalah, dengan menggunakan properti flex-direction, kita dapat mengubah arah susunan elemen, misalnya menjadi kolom. Jika menggunakan nilai column, elemen akan memanjang ke bawah. Jika menambahkan reverse, maka urutan elemen akan dibalik.

Kesimpulan skenario 4 adalah, dengan mengaktifkan flex-wrap: wrap;, elemen akan turun ke baris berikutnya saat ruang tidak cukup. Namun jika menggunakan nowrap, elemen akan tetap berada dalam satu baris dan tidak turun.

Kesimpulan skenario 5 adalah, dengan menggabungkan beberapa properti flex, kita dapat menciptakan kombinasi layout yang fleksibel. Meskipun demikian, perlu diperhatikan bahwa beberapa properti mungkin tidak berfungsi karena tertimpa oleh aturan CSS lainnya.

Kesimpulan skenario 6 adalah, dengan menambahkan properti justify-content, kita bisa mengatur posisi elemen di dalam kontainer flex, seperti meratakannya ke kiri, tengah, atau kanan.
