Kesimpulan 1: Di dalam sebuah container grid, kita bisa menambahkan elemen lain yang juga diatur menggunakan grid, misalnya dengan elemen div bernama item_grid.
Kesimpulan 2: Dengan mengubah properti display menjadi inline-grid, elemen tetap bisa tersusun sejajar dengan elemen lain dalam satu baris, selama ruangnya memungkinkan.
Kesimpulan 3: Properti gap memungkinkan kita mengatur jarak antar elemen. grid-column-gap digunakan untuk mengatur jarak horizontal (antar kolom), sedangkan grid-row-gap untuk jarak vertikal (antar baris).
Kesimpulan 4: Penggunaan grid-gap pada item grid membantu memberikan jarak yang konsisten antara baris dan kolom dalam layout grid.
