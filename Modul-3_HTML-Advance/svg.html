SVG (Scalable Vector Graphics) adalah format gambar berbasis XML yang digunakan untuk menggambar grafik vektor. SVG memungkinkan Saya untuk membuat gambar yang dapat diskalakan tanpa kehilangan kualitas. Dalam contoh ini, Saya telah membuat beberapa bentuk dasar menggunakan SVG. Saya menambahkan beberapa bentuk seperti lingkaran, persegi panjang, dan poligon.
